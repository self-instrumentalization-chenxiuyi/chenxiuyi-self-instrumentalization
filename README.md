# Self-Instrumentalization Theory (自器化理论)

**一个用于批判性分析当代社会的新型理论框架**。它揭示并命名了现代个体将自身无休止地工具化为“可优化资源”的系统性过程。

## 核心论点
现代性的深层困境，并非简单的“异化”或“剥削”，而是一种主动的、系统性的 **“自我工具化”**。个体在社会、技术与资本的递归逻辑驱动下，持续将自身拆解为可度量、可比较、可交易的碎片，并依据外部锚点进行调节与优化，从而陷入“为优化而优化”的循环牢笼。

## 核心模型：拆解-锚定-调节循环
这是驱动“自器化”的核心动力学模型：
1.  **拆解**：将完整的生活、情感、身体与潜能分解为离散、可量化的指标。
2.  **锚定**：为这些指标设定外部的、量化的价值标尺（如效率、评分、吸引力数据）。
3.  **调节**：个体在外部反馈与内部驱动下，调整行为以符合标尺，其新数据又反馈回系统，开启新一轮拆解，形成递归循环。

## 应用与讨论
该框架可用于分析工作伦理、情感资本主义、绩效社会、数字自我呈现、智能技术治理等诸多现象。
